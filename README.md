Expense Tracker (Full Stack)

A simple full-stack Expense Tracker web app built using FastAPI, JWT Authentication, and Supabase, with a frontend using HTML, CSS, and JavaScript.

✨ Features
User Signup & Login (JWT Auth)
Add & View Expenses
Category-wise tracking
Monthly budget management
Pie chart visualization
🛠 Tech Stack
Frontend: HTML, CSS, JavaScript, Chart.js
Backend: FastAPI, Uvicorn, JWT
Database: Supabase
